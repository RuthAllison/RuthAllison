The Data Professional - Reflections

This module has been an incredibly steep learning curve for me as I have no background in coding. Although I have completed a few basic Python lessons I have never formally learned any coding since BBC Basic back in the 1980s. Just creating this portfolio felt like a huge achievement - I'm really proud of myself for having got to grips with enough html to have multiple pages with links between them. Codio has been good for providing a platform for practice but the most useful website I have come across is w3schools.com

In the space of 12 weeks I have had to get to grips with Python, R, SQL and html. This ePortfolio is my first foray into html and I'm pleased with how it looks. Although I would struggle to create a page from nothing, I am confident in my ability to take a template and personalise it - I understand enough of the code to do that.

I think that I came into this course not really knowing what to expect - the amount of computing I was going to need to learn was a shock.
